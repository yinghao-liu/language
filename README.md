# language
## 总述



一个程序由以下部分组成

- 注释
- 变量
- 表达式
- 函数
- 类
- 接口
- 枚举

### 注释

单行注释

```
//
```

多行注释，不能嵌套

```
/*
...
*/
```



### 函数

```
func name(arg int) int
```

### 类

```
class name:base {

}
```

### 接口

```
interface name:base {

}
```

### 枚举

```
enum name:base {

}
```
